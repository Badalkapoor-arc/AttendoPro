About Attendance System

📌 Project Overview

This project automates attendance tracking using a fingerprint sensor.

🎯 Objective

To automate the attendance process and eliminate manual errors.

🔧 How It Works

  • Teacher Fingerprint: Starts countdown for attendance.
  • Student Fingerprint: Marks attendance.

💻 Technologies Used

  • Arduino, Fingerprint Sensor
  • HTML, CSS, JavaScript, Arduino Code
  • Google Gemini API, Firebase (Future)

🚀 Future Scope

  • More detailed storing of data
  • More encryption settings
  • Notification System
